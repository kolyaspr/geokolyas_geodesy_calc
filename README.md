## О проекте

**GeoKolyas** — мобильное приложение **«Геодезический калькулятор»**, разработанное на Flutter в рамках практической работы по предмету **«Основы разработки мобильных приложений»**.

Приложение предназначено для перевода угловых величин и решения базовых геодезических задач. Основной фокус работы — разделение пользовательского интерфейса и вычислительной логики, обработка пользовательского ввода, адаптивная вёрстка и сборка Android APK.

Реализованы четыре вычислительные функции:

1. перевод градусов из формата **ГГ° ММ′ СС″** в десятичные градусы;
2. перевод **десятичных градусов** в формат **ГГ° ММ′ СС″**;
3. решение **прямой геодезической задачи**;
4. решение **обратной геодезической задачи**.

- главный экран с навигацией по четырём функциям;

### Принцип разделения логики

Проект разделён на две основные части:

- **UI-слой** — экраны приложения находятся в папке `app/lib/screens/`;
- **вычислительная логика** — класс `GeoMath` находится в файле `app/lib/utils/geo_math.dart`.

Такой подход упрощает поддержку проекта: интерфейс можно менять отдельно от формул и вычислений.

## Реализованные вычисления

### 1. ГМС → десятичные градусы

```text
D = ГГ + ММ / 60 + СС / 3600
```

---

### 2. Десятичные градусы → ГМС

```text
ГГ = целая часть градуса
ММ = целая часть минут
СС = остаток, переведённый в секунды
```

---

### 3. Прямая геодезическая задача

По исходным координатам точки `A`, расстоянию `D` и дирекционному углу `α` вычисляются приращения и координаты точки `B`.

```text
ΔX = D · cos(α)
ΔY = D · sin(α)

Xb = Xa + ΔX
Yb = Ya + ΔY
```

---

### 4. Обратная геодезическая задача

По координатам двух точек `A` и `B` вычисляются расстояние, приращения координат, румб, дирекционный угол и четверть направления.

```text
ΔX = Xb - Xa
ΔY = Yb - Ya
D  = √(ΔX² + ΔY²)
```

## Как запустить проект

### 1. Клонировать репозиторий

```bash
git clone https://github.com/kolyaspr/geokolyas_geodesy_calc.git
cd geokolyas_geodesy_calc/app
```

---

### 2. Установить зависимости

```bash
flutter pub get
```

---

### 3. Проверить доступные устройства

```bash
flutter devices
```

---

### 4. Запустить приложение

На выбранном устройстве или эмуляторе:

```bash
flutter run
```

Если нужно запустить именно на Android-эмуляторе с конкретным ID:

```bash
flutter run -d emulator-5554
```

Если идентификатор эмулятора отличается, его нужно взять из вывода команды `flutter devices`.

---

## Как собрать APK

Команды выполняются из папки `app/`.

```bash
flutter clean
flutter pub get
flutter build apk --release
```

Готовый APK после сборки появится по пути:

```text
app/build/app/outputs/flutter-apk/app-release.apk
```

Если нужно скопировать свежий APK в корень репозитория:

```bash
copy build\app\outputs\flutter-apk\app-release.apk ..\app-release.apk
```

---

## Как установить APK на Android

### Способ 1. Скачать готовый APK из репозитория

В корне репозитория уже находится файл:

```text
app-release.apk
```

Его можно скачать, передать на Android-устройство и открыть для установки.

При установке Android может попросить разрешить установку из неизвестных источников.

---

### Способ 2. Собрать APK самостоятельно

```bash
cd geokolyas_geodesy_calc/app
flutter build apk --release
```

После сборки файл будет находиться здесь:

```text
build/app/outputs/flutter-apk/app-release.apk
```

---

### Способ 3. Установить через ADB

Если устройство подключено по USB или запущен эмулятор:

```bash
adb install build/app/outputs/flutter-apk/app-release.apk
```

При повторной установке:

```bash
adb install -r build/app/outputs/flutter-apk/app-release.apk
```
